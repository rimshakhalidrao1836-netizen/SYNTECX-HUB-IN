# NumPy Data Explorer

A simple Python project to demonstrate NumPy basics, performance, and file handling.

## What it does
- Creates and manipulates NumPy arrays
- Demonstrates indexing, slicing, and broadcasting
- Compares speed and memory usage of NumPy vs Python lists
- Saves and loads arrays using `.npy` files

## Requirements
Python 3.8+  
Install: `pip install numpy`
